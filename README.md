# Ground Engineering Toolbox

A web-based toolbox for practical and research-oriented applications in:

- Geothermal engineering
- Offshore anchor engineering

## Modules

### Geothermal
- ATES screening
- BTES preliminary sizing
- Thermal property tools

### Offshore Anchor
- Anchor capacity screening
- Load–displacement estimation
- Ground improvement effects
